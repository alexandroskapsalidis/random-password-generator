This project is a simple Random Password Generator built with HTML, CSS, and JavaScript.
The user can customize the generated password by choosing its length and selecting which
types of characters to include:
- lowercase
- uppercase
- numbers
- symbols
Once the options are set, clicking the “Generate” button produces a secure random password instantly.
